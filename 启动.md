后端和前端可以分别开两个终端来跑。

后端在项目根目录 C:\Users\jiayan\Desktop\RAG 启动：

```
cd C:\Users\jiayan\Desktop\RAG 
conda activate RAG_123
python -m uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000 
```

启动后一般访问：

- 后端服务: [http://127.0.0.1:8000](http://127.0.0.1:8000/)
- 接口文档: http://127.0.0.1:8000/docs

前端看你的项目结构是 frontend\app.py，而且 CORS 也配了 8501，所以前端应该是 Streamlit。第二个终端这样启动：

```
cd C:\Users\jiayan\Desktop\RAG streamlit run frontend\app.py --server.port 8501 
```

启动后访问：

- 前端页面: [http://127.0.0.1:8501](http://127.0.0.1:8501/)

如果 streamlit 命令不可用，就用：

```
python -m streamlit run frontend\app.py --server.port 8501 
```

建议启动顺序：

1. 先起后端
2. 再起前端
3. 打开前端上传文档测试
